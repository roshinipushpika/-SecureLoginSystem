# Secure Login System

A secure login web application built using Flask and SQLite.

## Features

- User Registration
- Secure Login
- Password Hashing using bcrypt
- Session Management
- Logout Functionality
- Input Validation
- SQL Injection Protection

## Technologies Used

- Python
- Flask
- SQLite
- Flask-Bcrypt
- HTML
- CSS

## How to Run

Install packages:

bash
py -m pip install -r requirements.txt


Run database:

bash
py database.py


Run application:

bash
py app.py


Open:

text
http://127.0.0.1:5000


## Expected Outcome

The system securely stores hashed passwords, validates user input, protects against SQL injection, and manages authenticated user sessions safely.
